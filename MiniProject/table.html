HTML - Tables

 

What is a Tables?

HTML Table is an arrangement of data in rows and columns, or possibly in a more complex structure.

An HTML table is defined with the “table” tag. Each table row is defined with the “tr” tag. A table header is defined with the “th” tag. By default, table headings are bold and centered. A table data/cell is defined with the “td” tag.

Syntax:

<table>
  <tr>
     <th>Firstname</th>
     <th>Lastname</th>
     <th>Age</th>
  </tr>
  <tr>
     <td>Matt</td>
     <td>Silvestre</td>
     <td>28</td>
  </tr>
  <tr>
     <td>Jessica</td>
     <td>Alas</td>
     <td>22</td>
  </tr>
  <tr>
     <td>Kim</td>
     <td>Salas</td>
     <td>25</td>
  </tr>
</table>
   
Output:
Firstname Lastname Age
Matt Silvestre 28
Jessica Alas 22
Kim Salas 25
